Short description:


Filter nodes: show only main search results / show main search results + their references / show main search results + citing papers / show all
Filter edges by weight: does not apply for Coauthor and Citation neworks. Filters links based on the respective frequencies.
Network type: choose the network you want to visualize. CoAuthor: paper are connected if they share one or more authors; Citation: papers are connected if one cites the other; BiblioCoupling: paper are connected if they share one or more references; CoCitation: paper are connected if they share one or more citing papers; Keywords: paper are connected if they share one or more keywords.
Filter nodes by degree: filter nodes based on the current number of links connected to them.
Run community detection: recolor nodes based on a Louvain community detection algorithm.
Search for: by pressing "highlight" you can highlight nodes which have the respective search term in their label.
Hover over a node: Get more information.
Single click on a node: highlight the node and its peers.
Double click on a node: Use DOI to find the paper.
Click anywhere: Reset colors.
